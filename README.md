# Allowance Request Form

เว็บแอปพลิเคชันสำหรับสร้างใบขอรับเงินค่าตอบแทนเบี้ยเลี้ยงเหมาจ่ายสำหรับเจ้าหน้าที่

## วิธีการใช้งาน

1. กรอกชื่อและตำแหน่ง
2. เลือกเดือนเริ่มต้น
3. คลิก "สร้างใบขอรับเงิน" เพื่อสร้างใบขอรับเงินสำหรับ 12 เดือนถัดไป

## เทคโนโลยีที่ใช้

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS

## การรันโปรเจกต์

```bash
npm install
npm run dev
```

เปิด http://localhost:3000 ในเบราว์เซอร์
